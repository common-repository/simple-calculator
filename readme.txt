This is a simple calculator plugin for calculations to be made in admin panel as well as in the front end.

== Description ==

Simple Calculator Plugin is a calculator that can be used from the admin panel.This is useful when any user needs calculations to be done when they are on the admin panel as well as this plugin is now working with the front end as well.

Just insert a simple shortcode into your page or posts and you are ready to use this superb calculator.

You will get the results on the same page of the calculator addition, subtraction, multiplication and division all at one place. this is the first release of this plugin and it is really very easy to use just install it and click the scalculator link in the settings and this simple ajax calculator is ready for use.

<strong>This Calculator is build for Admin use as well as on the front end use.<br /> Simply use [scalculator] Short code anywhere in the post or page and Simple calculator is ready to use. Unique CSS Class are already defined with the help of which you can design your calculator as per requirement.</strong>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `simple-calculator` folder after extracting the zip file to the `/wp-content/plugins/` directory or simply upload the zip file from the admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to setting -> scalculator, and its ready to go..!
